<I/E 구분>
1. 내가 느와르 영화에 등장한다면
    -> "내가 나서서 직접 모든 일을 해결한다!" 마피아 조직의 두목 = E
    -> "나서기는 싫어" 명석한 두뇌로 해결책을 제시하는 든든한 조력자 = I
2. 다른 조직과의 영역 침범 문제로 협상 장소에 나가게 된 당신.
   약속 장소에서 상대 조직의 후계자이자 당신의 이상형을 만나게 되는데...
    -> "너네 구역을 넓혀주는 대신 조건이 있지" 곧바로 연락처를 알아낸다 = E
    -> "아무리 그래도 다른 조직인데..." 혼자 조용히 짝사랑한다 = I

<N/S 구분>
1. 사랑하는 연인이 다른 조직에서 쏜 총에 맞아 나 대신 목숨을 잃었을 때
    -> "어떻게 그럴 수가 있지?" 복수를 다짐하며 이를 간다 = N
    -> "......" 큰 슬픔에 빠져 복수할 생각조차 들지 않는다 = S
2. 새로운 사업을 추진할 기회를 맞딱뜨렸을 때 나는
    -> 리스크를 감수해야 하더라도 새로운 사업을 시도해본다 = N
    -> 했다가 실패하면 어떡해? 기존에 추진하던 사업에 열중한다 = S
3. 조직의 사업을 확장하기 위해 투자자들을 유치하고자 할 때 나는
    -> "우리 회사가 지난 해 달성한 기록들을 보면..."
        우리 조직의 업적에 대해 수치를 들어 자세하게 설명한다 = S
    -> "우리 회사의 신뢰도에 대해 말하자면..."
        비유를 사용해 투자자들을 설득한다 = N

<F/T 구분>
2. 부하가 뛰쳐들어와 "다른 조직하고 시비가 붙어서 일이 커졌다"고 말할 때 나는
    -> "우리쪽 애들은? 크게 다친 애들 없어?" 부하들 걱정부터 한다 = F
    -> "시비가 왜 붙었는데?" 사건의 전후관계부터 파악한다 = T
3. 야심차게 준비했던 사업이 결국 실패로 돌아갔을 때 나는
    -> "비록 실패했지만 좋은 사람들을 많이 알게됐으니 됐어."
        실패는 했지만 그 과정 속에서 교훈을 얻었다고 생각한다 = F
    -> "이 사업은 시작하지 말았어야 하는건데..."
        실패했다는 사실때문에 화가 난다 = T

<J/P 구분>
1. 다른 조직으로부터 수상하지만 솔깃한 제의가 들어왔을 때 나는
    -> 문제가 생기면 그때가서 생각하자. 곧바로 제의를 받아들인다 = P
    -> 무슨 속셈이 있는지 철저하게 조사하고 계획을 짜둔 뒤 결정한다 = J
2. 믿었던 조직원으로부터 배신 당했다는 사실을 알았을 때 나는
    -> "어떻게 그럴 수가 있지?" 어떻게하면 복수를 할지 생각한다 = P
    -> "원래 이 바닥은 믿을 놈 하나 없지." 체념하고 다른 일에 몰두한다 = J
3. 우리 조직에 스파이가 있다! 스파이인 것이 들통난 그는 자신의 조직을 배신하고
    우리 조직으로 받아들여 달라고 애원하는데...
    -> 한 번 조직을 배신한 자는 또다시 배신하게 되는 법. 절대 받아주지 않는다 = J
    -> 이중 스파이로서 받아들여 주지. 하지만 끝까지 의심한다. = P
